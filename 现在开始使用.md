```

```

```

```

## 📋 日常使用流程

### 每天开始工作时

1. **启动服务**（如果未运行）
   ```bash
   ./start-service.sh start
   ```

2. **打开Chrome和网站**
   - 打开Gmail或163mail

3. **使用Claude Desktop**
   - 直接开始使用工具

### 结束工作时

可以选择：
- **保持服务运行**（推荐）- 下次直接使用
- **停止服务**
  ```bash
  ./start-service.sh stop
  ```

## 🔍 监控和调试

### 查看服务状态

```bash
./start-service.sh status
```

### 实时查看日志

```bash
./start-service.sh logs -f
```

按 `Ctrl+C` 退出日志查看。

### 重启服务

如果遇到问题：

```bash
./start-service.sh restart
```

## 💡 使用技巧

### 1. 多个网站同时使用

可以同时打开Gmail和163mail，工具会自动注册：

```bash
./start-service.sh logs | grep Registered
```

你会看到多个tab的工具注册。

### 2. 查看工具详情

在Claude Desktop中：

```
请详细说明search_emails工具的用法
```

### 3. 组合使用工具

```
先搜索包含"项目"的邮件，然后获取未读邮件
```

## 🐛 遇到问题？

### 问题1：工具列表为空

**解决：**
1. 确认服务正在运行：`./start-service.sh status`
2. 确认已打开Gmail或163mail
3. 刷新网页
4. 重启Claude Desktop

### 问题2：工具调用失败

**解决：**
1. 查看实时日志：`./start-service.sh logs -f`
2. 在Claude Desktop中调用工具
3. 观察日志中的错误信息

### 问题3：服务无法启动

**解决：**
```bash
# 清理端口
lsof -ti :3711 | xargs kill -9

# 重新启动
./start-service.sh start
```
